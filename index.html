Our Journey

As Omdena Cameroon Local Chapter, we have automated land use and land cover mapping using computer vision and satellite imagery to better monitor the environment, prevent conflicts, and support long-term climate adaptation efforts in Cameroon. Through this initiative, we built a machine learning model that can classify Land Use and Land Cover categories in satellite imagery. Addionally, we've generated LULC map for Cameroon, and created a web GIS dashboard to display the results. Our open-source dataset and code are available on GitHub, and we hope that they will continue to be useful for other researchers and organizations.
